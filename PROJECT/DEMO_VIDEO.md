# Demo Video (Required for submission)

Per the assignment guidelines, place your **demo video** in the project root folder.

- **Filename:** e.g. `demo.mp4` or `demo.webm`
- **Content:** The video must explain all features implemented and show a working demo of the features, plus a walkthrough of both the Frontend and Backend codebases.

Failing to submit this will lead to immediate disqualification.

After recording, add the file to the repo and commit.
